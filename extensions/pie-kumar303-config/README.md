# pie-kumar303-config

Extension manager for the `pie-kumar303` package. Run `/pie-kumar303-config` to selectively install or remove extensions from this repo.

## Usage

Type `/pie-kumar303-config` to open a two-panel UI:

- **Left panel**: Checkbox list of available extensions. Use ↑↓ to navigate, Space to toggle.
- **Right panel**: README preview for the highlighted extension. Use g/G for top/bottom, d/u for page up/down.

Press **Enter** to apply changes (creates/removes symlinks in `~/.pi/agent/extensions/`).  
Press **Escape** to cancel without changes.
